1) Написать cd <имя деректории>.
2) ~ - сокращённое обозначение домашнего каталога пользователя. . - текущий каталог. .. - каталог выше.
3) rm - команда удаления файла или папки..
4) cd - команда перехода в деректорию. 
5) cp - команда копирования.
6) mv - команда перемещения/переименовования файлов.
7) mv <старое название> <новое название>.
8) mv <название файла/папки> <куда переместить>.
9) -R - рекурсия.
10) -r - то же самое, а так же реверсирование при сортировке.
11) git - система контроля версий и для работы над одним проектом с разных ПК.
12) Команда инициализации локального репозитория.
13) git branch <название ветки>.
14) git checkout <название ветки>.
15) Слияние - это слияние файлов и коммитов из 2х веток в одну.
16) git rm - команда удаления файла из отслеживаемых файлов(из индекса).
17) git revert - это отмена существующих коммитов.
